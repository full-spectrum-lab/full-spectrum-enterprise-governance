# Examples

This directory contains public preview example materials for enterprise governance discussion.

## Included now

### `fshi-api-contract/`

Minimal inspection-chain samples showing:

- request
- response
- RiskAlert
- AuditTrace
- human-readable report

### `synthetic-cases/`

Synthetic public case notes carried forward from earlier protocol exploration.

They are useful for:

- scenario framing
- case review
- protocol discussion

They should not be treated as production validation records.
